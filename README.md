# Personal AI Data Analyst

An AI-powered data analysis application built with Streamlit.

## Features

- Upload CSV, Excel, or JSON files
- Automatic dataset summary
- Smart analysis suggestions
- Interactive visualizations
- Local LLM support using Ollama
- Download analysis results

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- DuckDB
- Ollama (Optional)

## Installation

```bash
git clone https://github.com/yourusername/Personal-AI-Data-Analyst.git

cd Personal-AI-Data-Analyst

pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
